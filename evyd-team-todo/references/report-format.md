# 汇总报告格式

## 全员汇总模板

```
📊 团队待办汇总 YYYY-MM-DD

张三：N条未完成（M项逾期）
 - [ ] 任务A | Due:2026-04-14 ⚠️
 - [ ] 任务B | Due:2026-04-20
 - [ ] 任务C

李四：N条未完成（M项逾期）
 - [ ] 任务D | Due:2026-04-13 🔴

─────────────────
全员统计：X条未完成 / Y项逾期

## 排序规则

每个成员的待办按 Due 日期从近到远排序，无 Due 的放最后。
```

## 标记规则

标记根据 Due 字段与当前日期**动态计算**，不存储在数据文件中：

| 符号 | 条件 |
|---|---|
| 🔴 | Due = 今天（紧急） |
| ⚠️ | Due = 明天（预警） |
| 🔥 | Due < 今天（逾期） |
| 无标记 | Due > 明天，或无 Due |

- 已完成的 TODO 已从数据文件中删除，**不出现在报告中**
- 无"已完成"计数——完成即消失

## 逾期判断

截止日期早于当前日期且仍存在于文件中的条目视为逾期。
