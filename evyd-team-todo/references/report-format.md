# 汇总报告格式

## 全员汇总模板

```
📊 团队待办汇总 YYYY-MM-DD

张三：N未完成 / M已完成
 - [ ] 任务A（高优，明天截止⚠️）
 - [ ] 任务B（中优）
 - [x] 任务C ✅

李四：N未完成 / M已完成
 - [ ] 任务D（高优，今天截止🔴）

─────────────────
全员统计：X未完成 / Y已完成 / Z项逾期
```

## 标记规则

| 符号 | 含义 |
|---|---|
| 🔴 | 今天截止（紧急） |
| ⚠️ | 明天截止（预警） |
| ✅ | 已完成 |
| 无标记 | 截止日 >2 天或无截止日 |

## 逾期判断

截止日期早于当前日期且状态为 `- [ ]` 的条目视为逾期。
