# 数据格式规范

## team-todo.md 格式

```markdown
# Team TODO

## 姓名
- [ ] 任务内容 | 截止:04-15 | 来源:老板 | 创建:2026-04-13
- [ ] 任务内容 | 截止:04-18 | 来源:老板 | 创建:2026-04-13 | 优先级:高 | 备注:进度50%
- [x] 任务内容 | 截止:04-12 | 来源:老板 | 创建:2026-04-10 | 完成:2026-04-12
```

### 字段说明

| 字段 | 必填 | 格式 | 说明 |
|---|---|---|---|
| checkbox | ✅ | `- [ ]` / `- [x]` | 未完成 / 已完成 |
| 任务内容 | ✅ | 自由文本 | 不含 `|` 字符 |
| 截止 | 可选 | MM-DD | 截止日期 |
| 来源 | ✅ | 姓名 | 谁布置的 |
| 创建 | ✅ | YYYY-MM-DD | 创建日期 |
| 优先级 | 可选 | 高/中/低 | 不写默认中 |
| 完成 | 完成时填 | YYYY-MM-DD | 完成日期 |
| 备注 | 可选 | 自由文本 | 进度说明等 |

## members.md 格式

```markdown
# 团队成员

| 姓名 | userid | 角色 |
|------|--------|------|
| 赵立 | 149681207 | admin |
| 张三 | zhangsan | member |
```

- `userid`：企微/飞书等渠道的用户 ID，用于自动匹配身份
- `角色`：`admin`（管理员，可管理全员 TODO）或 `member`（成员，只能操作自己的）
