# 日报模板

## 使用说明

此模板用于生成 Ned 的每日工作报告（EVYD 日报）。

## 模板内容

```markdown
## 一、工作流水账

### 1. [会议/工作名称]
**核心内容**：[简要归纳核心内容，1-3 句话]
**来源**：[[文件名（不带.md）]]

### 2. [会议/工作名称]
**核心内容**：[简要归纳核心内容]
**来源**：[[文件名（不带.md）]]

### 3. [会议/工作名称]
**核心内容**：[简要归纳核心内容]
**来源**：[[文件名（不带.md）]]

## 二、下一步汇总

- [待办事项 1]
- [待办事项 2]
- [待办事项 3]
```

## 格式规范

### 文件名格式
- 日期格式：`YYYY-MM-DD`
- 日期后有一个空格：`2026-03-10 「SengKang」...`
- 引用时去掉 `.md` 后缀：`[[💼 2026-03-10 「SengKang」「Proposal」Finalising Sengkang proposal]]`

### 内容要求
1. **简洁**：每个会议 1-3 句话概括核心
2. **来源**：必须标注 Obsidian 链接格式 `[[文件名]]`
3. **无标题**：不需要 `# 📅 日报` 这样的大标题
4. **无分隔线**：不需要 `---`

### 输出文件
- 路径：`/Users/Li.ZHAO/我赵立的仓库/24-Work-周报汇总/`
- 文件名：`YYYY-MM-DD EVYD 日报.md`
- 示例：`2026-03-10 EVYD 日报.md`

## 触发关键词

- daily-report
- 日报
- 生成日报
- 写日报
- 今天的工作

## 汇报规则

1. **有内容**：「日报已生成！✅ 文件位置：[路径]」
2. **没内容**：「今天没有工作记录 📋」
3. **不相关工作**：「今天没有工作记录 📋」
